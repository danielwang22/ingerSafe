# IngreSafe v2.0 測試清單

## 1. 啟動流程

### 首次啟動
- [ ] 應用程式正常啟動
- [ ] 顯示語言選擇頁面
- [ ] 6 種語言正確列出（繁中、英、日、韓、泰、越）
- [ ] 國旗 emoji 正確顯示
- [ ] 選擇語言後 Radio 填充、背景變綠
- [ ] 未選中項目 pressed 時背景變灰
- [ ] 未選擇時「確認」按鈕半透明不可點擊
- [ ] 選擇後「確認」按鈕可點擊
- [ ] 點擊「確認」正確導航到報告列表頁
- [ ] 點擊「略過」自動偵測語系並導航
- [ ] 語言選擇已儲存，重啟不再顯示

### 安全警告
- [ ] 進入報告列表後自動顯示安全警告
- [ ] 兩個警告項目正確顯示（展示食物、有毒物質）
- [ ] SVG icon 正確渲染（scan-eye、toxic）
- [ ] 點擊「確認」關閉視窗
- [ ] 重啟 APP 後安全警告再次顯示
- [ ] 點擊「不再顯示」關閉視窗
- [ ] 重啟 APP 後安全警告不再顯示
- [ ] 「關於我們」→「查看安全提醒」可再次開啟

### 非首次啟動
- [ ] 跳過語言選擇，直接進入報告列表頁

### AppBar

### 報告列表
- [ ] 報告按年份、月份分組

- [ ] 年份後綴正確（中文「年」、日文「年」、英文無）

### 報告卡片

- [ ] 標題、摘要正確
- [ ] 風險等級標籤顏色與圖示正確

  - [ ] 中風險：橙色 + warning icon
  - [ ] 高風險：紅色 + danger icon
  - [ ] 不確定：灰色 + help icon
- [ ] 風險標籤文字依語言顯示
- [ ] 刪除：第一次點擊顯示確認提示
- [ ] 刪除確認文字依語言顯示
- [ ] 3 秒內再按確認刪除
- [ ] 卡片進入動畫流暢
- [ ] Footer 顯示編輯（重新分析）圖示按鈕
- [ ] 點擊編輯圖示開啟 UploadDialog，標題與說明已預填
- [ ] 重新分析後，列表不會出現重複卡片
- [ ] 重新分析後，同一張卡片內容更新

### 空狀態
- [ ] 無報告時顯示空狀態提示
- [ ] 提示文字依語言顯示

### 底部導航欄
- [ ] 三個按鈕正確顯示（拍照、相簿、關於）
