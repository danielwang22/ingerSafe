# IngreSafe v2.0 使用指南

## 快速開始

```bash
flutter run
```

## 首次啟動流程

### 1. 選擇語言

首次開啟 APP 會顯示語言選擇頁面：
- 點選想要的語言（繁體中文、English、日本語、한국어、ไทย、Tiếng Việt）
- 點擊「確認 / Confirm」
- 或點擊「略過（自動偵測語系）」，APP 會根據系統語言自動設定

語言選擇後不會再顯示。如需更改語言，可在報告列表頁 AppBar 右側的語言切換器調整。

### 2. 安全提醒

進入 APP 後會看到安全警告彈窗，提醒兩件重要事項：
- **完整展示食物**：拍照時所有食物都要拍到，不要蓋住。拍不到的用文字補充。
- **有毒物質**：AI 無法從照片辨識毒品或有毒添加物，請自行確認食物安全。

可選擇：
- **確認**：關閉，下次開啟仍會提醒
- **不再顯示**：永久關閉（可從「關於」→「查看安全提醒」再次開啟）

## 主要功能

### 報告列表頁

進入 APP 後的主畫面，顯示所有飲食分析報告。

- **AppBar**：IngreSafe Logo + 分析狀態 + 語言切換器
- **報告列表**：按年份、月份分組，每張卡片顯示食物照片、標題、風險標籤、摘要、日期
- **底部導航欄**：拍照 / 相簿 / 關於

### 新增飲食紀錄

三種方式開啟上傳對話框：
1. 底部「拍照」— 開啟相機拍攝
2. 底部「相簿」— 從相簿選擇多張照片
3. 年份旁的 ➕ 按鈕

上傳對話框中：
1. 預覽並管理照片（可新增/移除）
2. 填寫標題（例如：午餐便當）
3. 填寫說明（選填）— 補充食材資訊或想了解的問題，**會一起傳送給 AI 分析**
4. 點擊「開始分析」

提交後：
- 臨時報告立即出現在列表頂部（顯示「分析中」）
- AppBar 顯示 spinner 和「AI 分析中...」
- 可同時提交多筆分析，spinner 會等所有分析完成才消失
- 分析完成後報告自動更新

### 查看報告詳情

點擊任何報告卡片進入詳情頁：
- **食物照片**：單圖直接顯示，多圖可左右滑動輪播
- **風險等級**：帶顏色標籤（低風險/中風險/高風險/不確定）
- **分析內容**：Markdown 格式，支援粗體、列表、連結
- **參考文獻**：獨立區塊，綠色左邊框，可點擊開啟外部連結
- **免責聲明**：僅供參考，不構成醫療建議

### 重新分析報告

兩個入口：
- **報告列表卡片**：點擊卡片右下角編輯（update）圖示
- **報告詳情頁**：點擊右上角 AppBar 編輯圖示

開啟後：
1. 上傳對話框自動帶入原始標題、說明和圖片
2. 可修改任何欄位或更換圖片
3. 點擊「開始分析」— 原報告即時顯示「分析中」，AI 重新處理
4. 分析完成後報告自動更新（保留同一筆 ID）

### 刪除報告

**從報告列表：**
1. 點擊卡片右下角垃圾桶圖示
2. 出現「再按一次確認刪除」提示
3. 3 秒內再按一次確認刪除（3 秒後提示自動消失）

**從報告詳情頁：**
1. 點擊右上角 AppBar 垃圾桶圖示
2. 同樣的「再按一次確認刪除」機制
3. 確認後自動返回列表頁

### 關於我們

點擊底部導航「關於」按鈕：
- 應用程式介紹和功能說明
- 免責聲明
- 版本資訊
- 「查看安全提醒」— 可重新開啟安全警告

## 風險等級說明

| 等級 | 顏色 | 說明 |
|------|------|------|
| 低風險 | 綠色 `#3B9B73` | 一般安全，適合孕期食用 |
| 中風險 | 橙色 `#F5A623` | 建議注意，適量食用 |
| 高風險 | 紅色 `#D63333` | 建議避免，不適合孕期 |
| 不確定 | 灰色 `#667269` | 無法判斷，請諮詢醫師 |

## 資料儲存

- 所有資料儲存在本地裝置（SharedPreferences）
- 不會上傳任何個人資訊
- 卸載 APP 會清除所有資料

### 資料結構
```json
{
  "id": "時間戳記",
  "title": "報告標題",
  "summary": "摘要說明",
  "riskLevel": "safe/warning/danger/unknown",
  "imageUrls": ["圖片路徑1", "圖片路徑2"],
  "date": "ISO8601 日期時間",
  "fullAnalysis": "完整分析內容 (Markdown)",
  "userNote": "使用者補充說明（選填，注入 AI Prompt）"
}
```

## 常見問題

### Q: 如何更改語言？
A: 報告列表頁 AppBar 右側的語言切換器可隨時切換。

### Q: 安全警告按了「不再顯示」後想再看？
A: 底部導航「關於」→「查看安全提醒」。

### Q: AI 分析失敗？
A: 檢查網路連線。確認 `.env` 檔案中 API key 正確設定。

### Q: 照片無法顯示？
A: 確認照片路徑正確且檔案存在。

### Q: 可以同時分析多筆嗎？
A: 可以。在第一筆分析進行中可提交第二筆，spinner 會等所有分析完成才消失。
